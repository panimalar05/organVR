<h1 align="center"> OrganVR </h1>
An interactive WebXR-based educational application built using A-Frame that allows users to explore human organs in an immersive 3D environment. OrganVR provides detailed anatomical information through interactive hotspots, making learning more engaging and intuitive in both desktop and Virtual Reality experiences.

## Features
- Interactive 3D human organ models.
- Organ selection menu for seamless navigation.
- Invisible clickable hotspots for anatomical exploration.
- Dynamic information panel that updates based on user interactions.
- Dedicated organ exploration interface with a back navigation system.
- WebXR and VR headset compatible using A-Frame.
- Educational descriptions for different anatomical structures.

## Supported Organs
### Human Lungs
- Trachea
- Left Lung
- Right Lung

### Human Kidneys
- Kidney Body
- Renal Artery
- Renal Vein

## User Interaction
### Menu Screen
- Select an organ using the available buttons.
- Clicking an organ opens the exploration interface.
### Organ Exploration
- Click different anatomical hotspots to learn about their functions.
- The information panel updates dynamically based on the selected structure.
- Use the Back button to return to the main menu.
### VR Experience
- OrganVR is designed using WebXR technologies and can be experienced on compatible VR headsets supported by modern browsers.
